# travis-yml plugin for Jenkins

Run Jenkins build step using .travis.yml

## Development

Prerequisites Ruby and bundler
```
gem install bundler
```

For development and to see this plugin in a test Jenkins server:
```
$ bundle install
$ jpi server
```
